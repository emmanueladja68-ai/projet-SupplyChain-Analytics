# projet-SupplyChain-Analytics
Dashboard Power BI interactif pour l'analyse de la performance logistique et commerciale
#  Analyse de Performance Supply Chain - Projet 

##  Objectif du Projet
Ce tableau de bord interactif Power BI a été conçu pour optimiser le suivi logistique et l'analyse des ventes mondiales. Ce projet démontre ma capacité à transformer des données complexes en indicateurs de performance (KPIs) exploitables pour la prise de décision.

##  Contenu du Dashboard
* **Page 1 : Performance Logistique**
  * Suivi des retards de livraison par région et par catégorie.
  * Cartographie interactive des flux mondiaux.
* **Page 2 : Analyse des Ventes**
  * Évolution temporelle des volumes de ventes.
  * Répartition du chiffre d'affaires par marché (Market Share).

##  Compétences Techniques
* **Outils :** Power BI Desktop, GitHub.
* **Traitement de données :** Nettoyage et structuration des types de données (Dates, Numériques).
* **Visualisation :** Création de graphiques dynamiques et hiérarchisés.

##  Comment consulter le projet ?
1. Téléchargez le fichier `.pbix` présent dans ce dépôt.
2. Ouvrez-le avec **Power BI Desktop**.
